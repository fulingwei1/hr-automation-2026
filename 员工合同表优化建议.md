# 员工合同表优化建议

## 一、现状评估

### 现有字段（可用）
| 字段 | 状态 | 说明 |
|------|------|------|
| 第二次续签到期（5年） | ✅ | 可用于计算到期日 |
| 第二次离到期天数 | ✅ | 可用于预警判断 |
| 第三次续签（无固定期限） | ✅ | 可区分是否需续签 |

### 结论
**现有数据基本可以支持合同续签预警功能。**

为更好支持合同管理全流程，建议增加以下字段：

| 合同状态 | 区分当前状态 | 有效/待续签/已续签/已终止 |

| 新增字段 | 用途 | 示例值 |

## 四、预警规则设置

| 预警等级 | 条件 | 颜色标识 | 动作 |
|----------|------|----------|------|
| 已过期 | 到期天数 < 0 | 黑色 | 立即处理 |
| 紧急 | 0 ≤ 天数 ≤ 30 | 红色 | 本周内完成续签 |
| 预警 | 30 < 天数 ≤ 60 | 橙色 | 启动续签沟通 |
| 关注 | 60 < 天数 ≤ 90 | 黄色 | 列入月度计划 |
| 正常 | 天数 > 90 | 绿色 | 下月再关注 |

---

## 五、日期格式建议

当前日期格式为"2026年1月14日"（文本格式），建议改为：

| 当前格式 | 建议格式 | 优点 |
|----------|----------|------|
| 2026年1月14日 | 2026-01-14 或 2026/1/14 | Excel日期格式，可自动计算 |

**修改方法**：
1. 将日期列格式改为日期格式
2. "离到期天数"列使用公式自动计算：`=到期日期-TODAY()`
3. 每次打开表格自动更新天数

## 六、自动化建议

### Excel公式设置

在"第二次离到期天数"列使用公式（假设到期日期在H列）：
```excel
=IF(H2="", "", H2-TODAY())
```

在"预警等级"列使用公式（假设天数在I列）：
```excel
=IF(I2<0,"已过期",IF(I2<=30,"紧急",IF(I2<=60,"预警",IF(I2<=90,"关注","正常"))))
```

### 条件格式设置
- 紧急（红色）：天数 ≤ 30
- 预警（橙色）：30 < 天数 ≤ 60
- 关注（黄色）：60 < 天数 ≤ 90
