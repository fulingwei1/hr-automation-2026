## 二、使用方法

### 方式1：直接对话（推荐）

在Claude Code中直接描述需求，系统会自动调用相应Skill：

```
用户：帮我分析12月的考勤数据
AI：[自动调用 attendance-report skill]
```

### 方式2：提供文件

将需要处理的文件路径告诉AI：

```
用户：筛选 /Users/flw/简历文件夹/ 里的简历，岗位是Python工程师
AI：[自动调用 resume-screener skill]
```

## 三、各Skill详细使用说明

### 1. attendance-report（考勤统计）

**输入：** 企业微信导出的考勤Excel

**输出：**
- XX月考勤明细表.xlsx
- XX月考勤汇总表.xlsx
- XX月加班统计表.xlsx（用于工资核算）
- XX月餐补统计表.xlsx
- XX月考勤异常报告.xlsx

**使用示例：**
```
"分析这个考勤文件：/path/to/考勤数据.xlsx"
"统计1月考勤，计算加班和餐补"
```

### 2. social-security-check（社保对账）

**输入：**
- HR系统员工社保数据
- 社保局导出的缴费明细
- 公积金中心导出的缴费明细（可选）

**输出：**
- 社保对账报告.xlsx（一致/差异/漏缴）
- 公积金对账报告.xlsx
- 对账邮件内容.txt

**使用示例：**
```
"对账1月社保，HR数据在xxx，社保局数据在xxx"
"核对这两个文件的社保是否一致"
```

### 3. resume-screener（简历筛选）

**输入：**
- 简历文件夹（支持PDF/Word）
- 岗位要求/JD

**输出：**
- 候选人排名表.xlsx（含TOP10推荐）
- 候选人详细评分.xlsx
- 筛选摘要

**使用示例：**
```
"筛选这些简历，要求本科以上，3年Java经验，会Spring"
"帮我从50份简历里找出最匹配机械工程师的5个人"
```

### 7. contract-renewal（合同续签与转正提醒）【新增】

**公司规则：**
- 试用期：6个月
- 首次合同：3年
- 续签合同：5年
- 无固定期限条件：签订2次后

**输入：** 人事档案系统Excel（含员工信息表、员工合同表）

**输出：**
- 合同续签预警报告（60天/30天提前预警）
- 试用期转正提醒（30天/7天提前预警）
- 续签合同模板
- 转正通知模板
- 紧急处理清单

**使用示例：**
```
"检查合同续签情况"
"查看哪些员工需要转正"
"生成本月合同转正提醒报告"
"为郑琴生成续签合同"
```

**预警规则：**
| 类型 | 提前60天 | 提前30天 | 提前7天 |
|------|----------|----------|---------|
| 合同续签 | 预警 | 紧急 | - |
| 试用期转正 | - | 即将 | 紧急 |

## 五、常见问题

### Q1: Skills没有响应？
确保在描述中包含关键触发词，如"考勤"、"简历"、"入职"等。

### Q2: 数据格式不对？
Skills会自动识别常见列名，但建议使用上述标准模板。

### Q3: 输出文件在哪里？
默认保存在当前工作目录下的相应文件夹中。

### Q4: 可以批量处理吗？
可以！直接告诉AI需要处理的所有文件或任务。
